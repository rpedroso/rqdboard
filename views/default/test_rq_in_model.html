Calls a function in a model

In this example calls the function slow_fib() in models/jobs.py

Calling instance methods is only supported passing the call as a string. Example: 


        from rqw import RQW
        job = RQW().enqueue('mail.send', args=(['me@example.com']))
RQW

is a class that serves as a `proxy` to be able to call functions in models.
The rqw.py module can be found in modules/rqw.py
